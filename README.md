# 📊 Sales Dashboard Project (Excel + Power BI)

---

## 📌 Project Overview
This project analyzes e-commerce sales data and presents insights using Excel and Power BI.  
An interactive dashboard was created to track KPIs like Revenue, Orders, and Average Order Value (AOV), helping in data-driven decision-making.

---

## 🎯 Objectives
- Clean and prepare raw sales data  
- Build key performance indicators (KPIs)  
- Visualize sales trends and business performance  
- Create an interactive dashboard using Excel and Power BI  
- Identify insights for business improvement  

---

## 📂 Dataset Information
**Dataset Used:** Global Superstore Dataset  

### Key Columns:
- Order ID, Order Date, Ship Date  
- Customer Details  
- Region, Country, City  
- Category & Sub-Category  
- Sales, Profit, Quantity  
- Discount, Shipping Cost  
- Order Priority  

---

## 🛠️ Tools Used
- Microsoft Excel (Data Cleaning, Pivot Tables, Charts)  
- Microsoft Power BI (Dashboard, DAX Measures)  
- Optional: Python (Data Preprocessing)  

---

## 📊 Key KPIs
- **Total Revenue** = Sum of Sales  
- **Total Orders** = Count of Unique Order IDs  
- **Average Order Value (AOV)** = Revenue / Orders  

---

## 📈 Dashboard Features
- KPI Cards (Revenue, Orders, AOV)  
- Sales Trend Analysis (Line Chart)  
- Category-wise Sales (Bar Chart)  
- Region-wise Sales Performance  
- Interactive Filters (Region, Category, Year)  

---

## 🔍 Key Insights
- Technology category generates highest revenue  
- Sales vary across different regions  
- Monthly sales show fluctuations  
- Medium priority orders are most frequent

## 📌 How to Use
